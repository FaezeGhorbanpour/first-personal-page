Multi-modal Fake News Detection

Our research involves examining news stories published on social media and news agencies, and identifying fake news by analyzing their image and text, as well as by employing machine learning algorithms like for embedding image and text, concatenation vectors and learning specific and corresponding features of text and image.
Transfer Learning: We explored several approaches for embedding text and image, and found that conceptual and semantic embedding worked best, Huggingface’s BERT model for text and ViT model for image. In the news agencies text, Big Bird model was more relevant and accurate due to the length of the documents.
Multi-modal Learning: We evaluated different algorithms to fuse text and picture vectors, and found that the Constrative Loss function and concatenation worked better for short texts, and attention based fusion for lengthy texts.
Semi-supervised Learning: We are now working on semi-supervised approaches like pseudo-labeling to leverage vast amounts of unlabeled news for training.

Blockchain System

This project’s goal is to use blockchain to create financial interactions. Wallets represent each customer’s bank account and cash. The consumer establishes an account and asks a loan, the bank acts as a blockchain extractor and approves the loan, and the bank’s head sets the regulations and monitors the credit transactions. We were the best project in the class for this project, which was for the network security course.
Security measures: To ensure the safety of the blockchain network, this project has included many layers of security, including authentication using the SHA512 hash function, transaction RSA encryption, and a user authentication method.
Blockchain: The system functions like a network. Banks, consumers, and servers are all nodes in a network, and transactions are exchanged between them.
